# Flight Reservation Desktop App (Tkinter + SQLite)

This is a simple desktop application built using **Tkinter** for the GUI and **SQLite** for storing the reservation records.

## Features
- Book a flight (Create)
- View all reservations (Read)
- Edit an existing reservation (Update)
- Delete a reservation (Delete)
